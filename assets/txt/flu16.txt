== TTMIK 1.6 - What is this? This is … / 이거 뭐예요? 이거… ==
    O: 책이에요.
    E: It's a book.

    O: 저예요.
    E: It's me.

    O: 이거 책이에요.
    E: This is a book.

    O: 카메라예요.
    E: It's a camera.

    O: 이거 카메라예요.
    E: This is a camera.

    O: 커피예요.
    E: It's coffee.

    O: 이거 커피예요.
    E: This is coffee.

    O: 사전이에요.
    E: It's a dictionary.

    O: 이거 사전이에요.
    E: This is a dictionary.

    O: 이거 뭐예요?
    E: What is this?

    O: 이거 마이크예요.
    E: This is a microphone.

    O: 이거 핸드폰이에요.
    E: This is a cellphone.

    O: 이거 물이에요.
    E: This is water.

    O: 이거 커피예요?
    E: Is this coffee?

    O: 아니요. 이거 물이에요.
    E: No. This is water.

    O: 네. 맞아요.
    E: Yes, that’s right.
