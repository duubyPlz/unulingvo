== TTMIK 4.15 - Anytime, No mention, Not affected, Not to anybody, In any way, No such _, It's nothing / 아무 때나, 아무 말도, 아무렇지도 않다, 아무한테도, 아무렇게나, 아무런 -도 없다, 아무것도 아니다 (아무 Part 2) ==
* g -> give
* s -> short

    # {#RELATED Level 4 Lesson 11}: 아무나, 아무도, 아무거나, 아무것도

    # Meanings:
    # 1. 아무 때나: anytime
    # 2. 아무 (말|이야기)도: no word/mention
    # 3. 아무렇지도 않다: to be unaffected
    #   * 아무렇다: affected (Note: Not used like this, only used with negative)
    # 4. 아무한테도: not to anybody
    #   * 아무: who, 한테: to
    # 5. 아무렇게나: in any way/however you like
    # 6. 아무런_{noun}_도 없다: no such _{noun}_
    #   * 아무, 런: like (e.g. 그런 like that) + {noun} + -도 없어요.
    # 7. 아무것도 아니에요.: It's nothing (big at all). / No biggie.

    # 1
    O: 아무 때나
    E: anytime

    O: 아무 때나 한국어 공부할 수 있어요.
    E: You can study{s} Korean{s} anytime.

    # 2
    O: 아무 말도
    E: no word / no mention

    O: 아무 이야기도
    E: no mention / no word

    O: 아무 말도 하지 마세요. 
    E: Please don't say a word.

    # 3
    O: 아무렇지도 않다
    E: to be unaffected / to be alright

    O: 저는 아무렇지도 않아요.
    E: I am okay.
    F: _I_ am okay.

    # 4
    O: 아무한테도
    E: not to anybody / to nobody

    O: 아무한테도 주지 말고 혼자 먹어요.
    E: Don't share{g} this with anyone, and eat it alone.

    # 5
    O: 아무렇게나
    E: in any way / however you like

    # Please don't just do it any way you want.
    O: 아무렇게나 하지 마세요.
    E: Please don't just do it however you want.

    # 두다
    # 치우다
    O: 아무렇게나 두지 말고, 좀 치워요.
    E: Don't put things randomly(?) (lit: in any way), clean up a bit{s}. 🤔

    # 6
    # There is no _ of any kind.
    O: 아무런 _도 없어요.
    E: There is no such _.

    # The following two sentences have the same meaning:
    O: 아무런 소식도 없어요.
    E: There's no such news at all.

    O: 아무 소식도 없어요.
    E: There's no news at all.

    # There's nothing. / No biggie.
    O: 아무것도 아니에요.
    E: (Wanting to hide something) That's nothing. / No biggie.
