== TTMIK 4.30 - [Sentence Building Drill] #2 ==
* s -> short
* l -> long

    # # Sentences
    # ### 1
    # > 열 명 초대했는데, 아무도 안 올 수도 있어요.
    # > I have invited ten people{s}, but it's possible that no one will come.
    # ### 2
    # > 오늘은 어제보다 훨씬 따뜻한 것 같아요.
    # > I think today is much warmer than yesterday.
    #
    # ### 3
    # > 지금 카페에서 어제 산 책을 읽고 있어요.
    # > Right now I am in a cafe, reading a book that I bought yesterday.

    # ### Variants
    # 열 명 = ten people{s},
    # 한 명 = one people{s},
    # 두 명 = two people{s},
    # 세 명 = three people{s},
    # 
    # 초대했는데, = I have invited
    # 말했는데, = I have told
    # 조심했는데, = I was careful
    # 열심히 공부했는데, = I studied hard
    # 
    # 아무도 안 올 수도 있어요 = but it's possible that no one will come.
    # 아무도 안 올 거예요 = but no one will come.
    # 아무도 모를 거예요 = but no one will know.
    # 아무도 안 할 거예요 = but no one will do it.
    # 아무도 초대 안 할 거예요 = but no one will be invited.
    # but (someone) might not come.
    # but (someone) might not give it.
    # but it might not be so (the case).
    # but it might not be funny.

    O: 열 명 초대했는데, 아무도 안 올 수도 있어요.

    #
    # 어제보다 = compared to yesterday.
    # 지난주보다 = compared to last week.
    # 지난달보다 = compared to last month.
    # 작년보다 = compared to last year.
    # 이거보다 = compared to this one.
    # 한국어보다 = compared to Korean.
    # 
    # 훨씬 따뜻한 것 같아요. = I think it is much{s} warmer
    # 훨씬 좋은 것 같아요. = I think it is much{s} better
    # 훨씬 재미있는 것 같아요. = I think it is much{s} more interesting
    # 훨씬 따뜻해요. = It's much{s} warmer
    # 훨씬 좋아요. = It's much{s} warmer
    # 훨씬 어려워요. = It's much{s} (more) difficult

    O: 오늘은 어제보다 훨씬 따뜻한 것 같아요.
    E: I think it is much{s} warmer today compared to yesterday.
    F: I think it is much{s} warmer _today_ compared to yesterday.

    O: 오늘은 어제보다 훨씬 좋은 것 같아요.
    E: I think it is much{s} better today compared to yesterday.
    F: I think it is much{s} better _today_ compared to yesterday.
